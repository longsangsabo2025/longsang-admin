# Frontend — Agent Instructions

> React 18 + TypeScript + Vite + Supabase + Zustand + TanStack Query

## Architecture

- **Entry**: `main.tsx` → `App.tsx` (90+ lazy-loaded routes)
- **Auth**: Supabase Auth, role-based routing (`/admin/*`, `/manager/*`, `/mobile/*`)
- **State**: Zustand (UI state) + React Query (server state) + useState (local)
- **UI**: shadcn/ui + Radix UI + TailwindCSS
- **Path alias**: `@/` → `src/`

## Component Pattern (MUST FOLLOW)

```typescript
import { useState, useCallback } from 'react';

  readonly title: string;
  readonly onAction?: () => void;
}

export function MyComponent({ title, onAction }: MyComponentProps) {
  const [loading, setLoading] = useState(false);

  return (
    <Card>
      <CardHeader>
        <CardTitle>{title}</CardTitle>
      </CardHeader>
      <CardContent>
        <Button onClick={onAction} disabled={loading}>
          Action
        </Button>
      </CardContent>
    </Card>
  );
}
```

## Supabase Real-time Subscriptions

```typescript
import { supabase } from '@/lib/supabase';
import { useEffect } from 'react';

// Subscribe to table changes — ALWAYS clean up on unmount
useEffect(() => {
  const channel = supabase
    .channel('notifications')
    .on('postgres_changes', {
      event: '*',          // 'INSERT' | 'UPDATE' | 'DELETE' | '*'
      schema: 'public',
      table: 'notifications',
      filter: `user_id=eq.${userId}`,  // optional row filter
    }, (payload) => {
      // payload.new = new row, payload.old = old row
      handleChange(payload);
    })
    .subscribe();

  return () => { supabase.removeChannel(channel); };
}, [userId]);
```

### React Query + Real-time Hybrid

```typescript
// Use React Query for initial fetch + refetch, real-time for live updates
const queryClient = useQueryClient();

const { data } = useQuery({
  queryKey: ['notifications', userId],
  queryFn: () => fetchNotifications(userId),
});

useEffect(() => {
  const channel = supabase
    .channel('notifications-live')
    .on('postgres_changes', {
      event: 'INSERT', schema: 'public', table: 'notifications',
    }, () => {
      // Invalidate React Query cache → triggers refetch
      queryClient.invalidateQueries({ queryKey: ['notifications', userId] });
    })
    .subscribe();

  return () => { supabase.removeChannel(channel); };
}, [userId, queryClient]);
```

## Rules

1. **TypeScript strict** — no `any`, define interfaces for all props
2. **shadcn/ui first** — always check `components/ui/` before creating custom UI
3. **`cn()` for classes** — `import { cn } from '@/lib/utils'`
4. **Lazy load pages** — `const Page = lazy(() => import('./pages/Page'))`
5. **useCallback/useMemo** — for expensive computations and callbacks passed as props
6. **Error boundaries** — wrap major sections with `<ErrorBoundary>`
7. **i18n** — use `useTranslation()` for user-facing text
8. **ESM only** — `import/export`, NOT `require()`
9. **Clean up subscriptions** — always `removeChannel()` in useEffect return
10. **Invalidate, don't duplicate** — use `queryClient.invalidateQueries()` when real-time triggers change
