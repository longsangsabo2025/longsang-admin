## PHẦN 1: THƠ LÝ BẠCH (LÝ BẠCH CHÍNH GỐC)

### 1.1 Về Lý Bạch (701-762)

- **Biệt danh:** Thi Tiên (Tiên thơ), Tửu Tiên (Tiên rượu)
- **Thời đại:** Nhà Đường (thời hoàng kim thơ ca Trung Hoa)
- **Di sản:** ~1000 bài thơ còn lại
- **Phong cách:** Phóng khoáng, lãng mạn, bay bổng

### 1.2 Các bài thơ nổi tiếng nhất

### 1.3 Các chủ đề thơ Lý Bạch

| Chủ đề | Tần suất | Ví dụ |
|--------|----------|-------|
| **Trăng** | Rất nhiều | Tĩnh Dạ Tứ, Nguyệt Hạ Độc Chước |
| **Rượu** | Rất nhiều | Tương Tiến Tửu, Đối Tửu |
| **Thiên nhiên** | Nhiều | Vọng Lư Sơn, Tảo Phát Bạch Đế |
| **Ly biệt** | Nhiều | Hoàng Hạc Lâu, Tặng Uông Luân |
| **Cô đơn** | Nhiều | Nguyệt Hạ Độc Chước |
| **Nhớ quê** | Nhiều | Tĩnh Dạ Tứ |
| **Hiệp khách** | Một số | Hiệp Khách Hành |

### 1.4 Cấu trúc thơ Đường

**Thất ngôn tứ tuyệt** (7 chữ, 4 câu):
```
Câu 1: Khai (mở)
Câu 2: Thừa (tiếp)
Câu 3: Chuyển (biến)
Câu 4: Hợp (kết)
```

**Ngũ ngôn tứ tuyệt** (5 chữ, 4 câu):
```
Như Tĩnh Dạ Tứ
```

**Cổ phong** (tự do hơn):
```
Như Tương Tiến Tửu
```

## PHẦN 2: CÔNG THỨC CHẾ THƠ LÝ BLẠCK

### 2.1 Pattern chính

```
THƠ GỐC (Nghiêm túc cổ điển)
    ↓
3 CÂU ĐẦU: Giữ nguyên hoặc paraphrase
    ↓
CÂU CUỐI: TWIST hiện đại, Gen Z
```

### 2.2 Các kiểu twist

**Kiểu 1: Thay từ cuối**
```
Gốc: Cúi đầu nhớ cố hương
Chế: Cúi đầu nhìn bảng lương
```

**Kiểu 2: Đổi ngữ cảnh**
```
Gốc: Nước sông Hoàng từ trời đổ xuống
Chế: Deadline từ trời đổ xuống
```

**Kiểu 3: Gen Z hóa**
```
Gốc: Sáng còn đen như tơ, chiều đã bạc
Chế: Sáng còn full pin, chiều đã 1%
```

**Kiểu 4: Vấn đề đời thường**
```
Gốc: Nâng chén mời trăng sáng
Chế: Nâng phone lướt TikTok
```

### 2.3 Mapping chủ đề

| Thơ gốc về | Chế thành về |
|------------|--------------|
| Trăng | Màn hình phone |
| Rượu | Trà sữa, cafe |
| Nhớ quê | Nhớ giường, nhớ weekend |
| Ly biệt | Hết lương, hết nghỉ phép |
| Thiên nhiên | Điều hòa, máy lạnh |
| Cô đơn | FA, crush không reply |
| Hiệp khách | Chad, sigma |

## PHẦN 4: NGUỒN CẢM HỨNG THƠ HÀI

### 4.1 Thơ Đường nổi tiếng khác (để chế)

**Hoàng Hạc Lâu - Thôi Hiệu:**
```
Tích nhân dĩ thừa hoàng hạc khứ
Thử địa không dư Hoàng Hạc lâu
→ Người xưa cưỡi hạc vàng bay mất
  Nơi đây còn lại... deadline thôi
```

**Xuân Hiểu - Mạnh Hạo Nhiên:**
```
Xuân miên bất giác hiểu
Xứ xứ văn đề điểu
→ Ngủ xuân không biết sáng
  Chỗ chỗ nghe... alarm kêu
```

**Đăng Quán Tước Lâu - Vương Chi Hoán:**
```
Dục cùng thiên lý mục
Cánh thượng nhất tằng lâu
→ Muốn thấy xa ngàn dặm
  Leo thêm một tầng lầu
  → Muốn thấy lương cuối tháng
     Làm thêm... một tháng nữa
```

### 4.2 Ca dao Việt Nam (để chế)

**Gốc:**
```
Công cha như núi Thái Sơn
Nghĩa mẹ như nước trong nguồn chảy ra
```
**Chế:**
```
Deadline như núi Thái Sơn
OT như nước trong nguồn chảy ra
```

**Gốc:**
```
Đêm qua ra đứng bờ ao
Trông cá cá lặn, trông sao sao mờ
```
**Chế:**
```
Đêm qua ra đứng bờ ao
Check phone phone hết pin, check sao... WiFi mất
```

### 4.3 Tục ngữ chế

| Gốc | Chế |
|-----|-----|
| Có công mài sắt có ngày nên kim | Có công lướt TikTok có ngày hết pin |
| Uống nước nhớ nguồn | Xài tiền nhớ lương |
| Ăn quả nhớ kẻ trồng cây | Dùng WiFi nhớ kẻ đóng tiền |
| Đi một ngày đàng học một sàng khôn | Họp một ngày dài, học một... chẳng gì |

### 4.4 Meme formats phổ biến

**"Ngàn năm trước..." format:**
```
Ngàn năm trước, ta từng [hành động cao quý]
Ngàn năm sau, ta vẫn [vấn đề đời thường]
```

**"Ta ngộ ra..." format:**
```
Cổ nhân nói [triết lý cao siêu]
Ta ngộ ra điều này... [timing hài]
```

**"Thiên hạ..." format:**
```
Thiên hạ ai người không [vấn đề]
Chỉ có ta là [tệ hơn]
```

## PHẦN 5: THƯ VIỆN TỪ VỰNG

### 5.1 Từ cổ điển hay dùng

| Từ | Nghĩa | Dùng khi |
|----|-------|----------|
| Quân bất kiến | Bạn không thấy sao | Mở đầu dramatic |
| Cố nhân | Người xưa/bạn cũ | Nhắc chuyện cũ |
| Nghi thị | Ngỡ là | So sánh |
| Cử đầu | Ngẩng đầu | Hành động |
| Đê đầu | Cúi đầu | Hành động |
| Thiên hạ | Dưới trời | Nói chung chung |
| Ngàn năm | Thời gian dài | Tạo contrast |
| Tu luyện | Luyện tập | Hành động |

### 5.2 Từ Gen Z để twist

| Từ cổ | Thay bằng |
|-------|-----------|
| Minh nguyệt (trăng sáng) | Màn hình phone |
| Tửu (rượu) | Trà sữa, cafe |
| Cố hương (quê) | Giường, nhà |
| Bạch phát (tóc bạc) | Rụng tóc |
| Kim (vàng) | Tiền lương |
| Kiếm (gươm) | Laptop |
| Ngựa | Grab, xe máy |
| Thư (thư từ) | Tin nhắn, email |

### 5.3 Vần hay dùng

```
-ang: lương, sương, hương, đường, trường
-ơi: rồi, thôi, trời, người, nơi
-ay: bay, hay, ngay, say, tay
-ời: đời, trời, người, thời, nơi
-ong: không, trong, cùng, xong, hồng
```

## PHẦN 6: WORKFLOW TẠO THƠ CHẾ

### 6.1 Quy trình 5 bước

```
1. CHỌN THƠ GỐC
   → Thơ Đường nổi tiếng, dễ nhận ra
   
2. CHỌN CHỦ ĐỀ HIỆN ĐẠI
   → Vấn đề Gen Z relate được
   
3. TÌM ĐIỂM TƯƠNG ĐỒNG
   → Mapping cảm xúc/tình huống
   
4. GIỮ 3 CÂU, CHẾ CÂU CUỐI
   → Twist bất ngờ
   
5. KIỂM TRA
   → Vần? Hài? Relatable?
```

### 6.2 Prompt cho AI

```
Chế bài thơ [TÊN THƠ] của [TÁC GIẢ] theo phong cách Lý Blạck.

Yêu cầu:
- Giữ 3 câu đầu nguyên hoặc paraphrase nhẹ
- Câu cuối twist về [CHỦ ĐỀ]
- Giữ vần nếu có thể
- Tone: Nghiêm túc → bất ngờ hài

Output: Thơ gốc + Thơ chế + Giải thích
```
