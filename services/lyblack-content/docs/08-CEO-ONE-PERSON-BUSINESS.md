## 1. TƯ DUY CEO 2026

### 1.1 Mindset shifts

| Cũ (2020s) | Mới (2026) |
|------------|------------|
| Hustle = Success | Leverage = Success |
| Thuê người để scale | Dùng AI/systems để scale |
| Làm mọi thứ | Làm việc đúng |
| Hours worked | Output created |
| Perfect launch | Fast iteration |

### 1.2 Công thức One Person Business

```
REVENUE = (Audience × Trust × Offer) × Automation

Audience: Số người biết bạn
Trust: % tin tưởng bạn
Offer: Giá trị bạn cung cấp
Automation: Leverage từ AI/systems
```

### 1.3 CEO vs Employee mindset

```
EMPLOYEE: "Tôi làm việc cho tiền"
CEO: "Tôi xây systems tạo ra tiền"

EMPLOYEE: "Tôi bận"  
CEO: "Tôi có hệ thống"

EMPLOYEE: "Tôi cần học thêm"
CEO: "Tôi cần AI/tools làm việc này"
```

## 5. DECISION FRAMEWORK

### 5.1 Yes/No Matrix

```
Trước khi làm bất kỳ việc gì, hỏi:

□ Việc này có tạo ra tiền không?
□ Việc này có build audience không?
□ Việc này có thể automate không?
□ Việc này chỉ tôi làm được không?

≥3 YES → Làm
≤2 YES → Delegate/Automate/Skip
```

### 5.2 Priority Matrix

```
                    URGENT
                      │
         ┌───────────┼───────────┐
         │    DO     │  SCHEDULE │
IMPORTANT├───────────┼───────────┤NOT IMPORTANT
         │ DELEGATE  │   DELETE  │
         └───────────┼───────────┘
                      │
                  NOT URGENT
```

### 5.3 $100/hour Test

```
Tự hỏi: "Việc này đáng $100/giờ không?"

YES → Tự làm, focus 100%
NO  → Automate hoặc skip

Ví dụ:
- Viết content strategy → YES
- Edit video nhỏ → NO (AI làm)
- Reply comments → NO (batch + AI)
- Negotiate deal lớn → YES
```

## 8. RISK MANAGEMENT

### 8.1 Platform Risk

```
KHÔNG bao giờ phụ thuộc 1 platform

RULE: 70-20-10
- 70% focus main platform
- 20% secondary platform
- 10% email list (OWN)

Email list = Insurance policy
```

### 8.2 Burnout Prevention

```
WARNING SIGNS:
- Dread tạo content
- Skip days off
- Quality giảm
- Health issues

PREVENTION:
- 1 day OFF không thương lượng
- Content batching (không làm daily)
- Vacation every quarter
- Boundaries rõ ràng
```

### 8.3 Income Diversification

```
Target: Không có source nào > 50% income

Ví dụ healthy mix:
- Creator funds: 20%
- Brand deals: 30%
- Products: 30%
- Services: 20%
```

## 9. SCALE PLAYBOOK

### 9.1 Phase 1: Foundation (0-10K followers)

```
FOCUS:
- Find voice/character
- Test content formats
- Build daily habit
- Engage with community

KHÔNG làm:
- Chạy ads
- Bán products
- Collab lớn
```

### 9.2 Phase 2: Growth (10K-100K)

```
FOCUS:
- Double down winning content
- Launch simple product
- Build email list
- Strategic collabs

BẮT ĐẦU:
- Monetization nhẹ
- Brand inquiries
- Systems automation
```

### 9.3 Phase 3: Scale (100K+)

```
FOCUS:
- Multiple revenue streams
- Team/VA if needed
- Premium offerings
- Community building

OPTIMIZE:
- Profit margins
- Time freedom
- Passive income %
```
