## 1. TRENDS 2026 CẦN NẮM

### 1.1 Xu hướng chính

| Trend | Áp dụng cho Lý Blạck |
|-------|----------------------|
| **Authentic > Polished** | Giữ vẻ tự nhiên, không quá edit |
| **Short-form dominates** | Focus 15-30s videos |
| **Hooks trong 1-2s** | Mặt Lý Blạck + text gây tò mò |
| **Retention > Views** | Giữ người xem đến hết video |
| **Niche content wins** | Cổ trang + hài = unique niche |
| **Community building** | Reply comments, tạo inside jokes |

### 1.2 Algorithm 2026

**TikTok ưu tiên:**
- Watch time & completion rate
- Shares > Saves > Comments > Likes
- Reply comments trong 1h đầu
- 80% trending audio + 20% original

**Instagram Reels:**
- Carousels đang rise
- Stories giữ chân core audience
- Reels để reach mới

**YouTube Shorts:**
- Dẫn traffic sang long-form
- Monetization tốt hơn TikTok

## 4. HOOK FORMULAS

### 4.1 Visual hooks (1-2s đầu)

```
1. Close-up mặt Lý Blạck + expression kỳ lạ
2. Text gây tò mò: "Tiên nhân 1300 tuổi nói về..."
3. Contrast: Cổ trang setting + modern problem
4. Pattern interrupt: Unexpected opening
```

### 4.2 Text hooks hiệu quả

```
✓ "POV: Lý Blạck dạy về [vấn đề]"
✓ "Tiên nhân 1300 tuổi lần đầu thấy [điều gì đó]"
✓ "Ngàn năm tu luyện, ta ngộ ra..."
✓ "Điều ta ước được biết sớm hơn..."
✓ "Tại sao [vấn đề] làm khổ đệ tử?"
```

### 4.3 CTA cuối video

```
✓ "Follow để tu luyện mỗi ngày"
✓ "Đệ tử nào đồng ý, comment 🙏"
✓ "Save để ngộ sau"
✓ "Share cho đệ tử đồng môn"
```

## 11. CRISIS MANAGEMENT

### 11.1 Potential issues

| Issue | Response |
|-------|----------|
| Negative comments | Ignore hoặc reply nhẹ nhàng |
| Copyright claim | Dùng nhạc royalty-free |
| Content stolen | Watermark, report |
| Backlash | Pause, assess, address nếu cần |

### 11.2 Nguyên tắc

```
- KHÔNG phản ứng khi đang emotional
- Giữ character ngay cả khi drama
- Controversy có thể = attention
- Biết khi nào im lặng là vàng
```
