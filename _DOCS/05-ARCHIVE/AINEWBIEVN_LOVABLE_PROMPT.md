# PROMPT CHO LOVABLE AI - AINEWBIEVN LANDING PAGE

Create a stunning product landing page and marketplace showcase for AINewbieVN - Vietnam's first comprehensive AI ecosystem. The design should feel like browsing a premium streaming/gaming platform (similar to Twitch/Unity Connect) but for AI products and services, with heavy focus on visual content, product cards, and discovery experience.

## DESIGN PHILOSOPHY

This is NOT an app interface - this is a MARKETING LANDING PAGE that showcases AI products/services like a visual marketplace. Users should feel like they're exploring, discovering, and shopping for AI solutions through rich visual content.

## VISUAL STYLE & COLOR SCHEME

**Dark Theme Gaming/Streaming Platform Aesthetic:**

- Primary Background: Deep blue-black (#0f1419, #1a1f2e, #0a0d14)
- Accent Purple: #6C63FF, #8B5CF6 (for CTAs and highlights)
- Accent Cyan/Teal: #00D9FF, #06B6D4 (for active states)
- Card Backgrounds: Dark navy-gray (#1e2433, #252b3b)
- Text: White (#FFFFFF) headings, Light blue-gray (#A8B3CF) body
- Glassmorphism: Semi-transparent cards with backdrop-blur

**Visual Principles:**

- Image-first design: Every product/service needs compelling visuals
- Card-based layout: Everything presented as browsable cards
- Hover effects: Scale, glow, and smooth transitions
- Rich thumbnails: Product screenshots, demos, illustrations
